使用 Hexo 搭建博客

安装 Node.js

安装 Hexo

初始化博客

Note: blog 目录是博客的根目录，可以自定义。

生成静态页面&部署

Github 托管

创建 Github 仓库

在 Github 上创建仓库，仓库名称为 $username.github.io，其中 $username 是你的 Github 用户名。

配置 _config.yml

生成&部署

生成完成后，在浏览器中访问 $username.github.io 即可看到博客页面。

安装主题

从 Hexo 主题 中挑选一个自己喜欢的主题，然后安装。

编写博客

  1. 进入source/_posts目录，新建一个 markdown 文件，文件名格式为：$year-$month-$day-$title.md。
  2. hexo g 生成静态页面。
  3. hexo d 部署到 Github。

已有博客换新电脑

  1. 安装 Node.js。
  2. 安装 Hexo。
  3. git clone 博客代码。
  4. 撰写博客。
  5. hexo g 生成静态页面。
  6. hexo d 部署到 Github。
